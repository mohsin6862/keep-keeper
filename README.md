## Projects : KeenKeeper

Description: This projects is like friendship tracker. By using this site user can see their friends.

Technologies used: Nextjs, React, JavaScript, Tailwind CSS, DaisyUI, React Icons, Rechart, Toastify, Context API, Local Storage etc. 

Features: 
1. user can see their friends information.
2. user can interact with their friends via call,text,and video calls.
3. User can see their all activity from timeline page (like when they interact with their friends and by using which platform (text,call,video call))
4. In stats page user can see their interaction from a pie chart which looks so attractive.
